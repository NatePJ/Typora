#### 命令

查看环境中安装的应用版本

```javascript
conda list   
```



打开anaconda控制台，输入下面命令进入pytorch的conda命令环境

```javascript
conda activate pytorch
```



> conda install nb_conda

```javascript
jupyter notebook
```
